# Task-Manager
FlowTask is a sleek, interactive task management web application. It allows users to create, organize, track, and manage daily tasks with a modern, user-friendly interface.
A clean and simple Task Manager App built with **HTML, CSS, and JavaScript**.
## ✨ Features

- Add, edit, and delete tasks
- Mark tasks as complete
- Responsive design (works perfectly on mobile & desktop)
- Clean and modern user interface
- Search and filter tasks
- Due date support

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript
